  • Fuzzyness: foo~1 will return articles containing "foo" exactly and all permutations of "foo" within a one-char difference (e.g. "foz")
  • Wildcards: bar* will return articles containing "bar" and any subsequent character(s)
  • Title tags: title:baz will return articles containing "baz" in their titles
  • Operators: +quux -xyzzy will return articles containing "quux" but not "xyzzy"
  • Weights: corge^10 grault^2 will return articles containing "corge" and/or "grault" but "corge" will have 5x relevance
