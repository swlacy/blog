I would neither be able to create nor maintain this website easily without the hard work of other programmers and designers. The following list documents the significant frameworks, code, and assets that I used while designing this site. Assets not documented on this page are credited at their source locations.

## Site Generation

 - SSG: [Hugo](https://gohugo.io)
 - CSS framework: [Tailwind CSS](https://tailwindcss.com)

## Assets

 - Icons: [Lucide](https://lucide.dev) (fork of [Feather Icons](https://feathericons.com))

## JS Libraries

 - Icon loader: [Lucide](https://unpkg.com/lucide@latest/dist/umd/lucide.min.js)
 - Inline image zooming: [Zooming](https://unpkg.com/zooming@latest/build/zooming.min.js)
 - Front page typing animation: [Typed](https://unpkg.com/typed.js@latest/dist/typed.umd.js)

## Styling

The theme, [Port](https://github.com/lacysw/hugo-port-theme), is my own design, but with influence from many others. The syntax highlighting is a slightly edited version of the [solarized-dark256](https://xyproto.github.io/splash/docs/solarized-dark256.html) Chroma style.
