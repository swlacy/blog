Source code of my blog, *Sid's Blog*, hosted at [swlacy.com](https://swlacy.com). Built using Hugo, TailwindCSS, and my custom theme, [Port](https://github.com/lacysw/hugo-port-theme).

Usage (requires Hugo in path):
```bash
$ git clone https://github.com/lacysw/blog.git && cd blog
$ git clone https://github.com/lacysw/hugo-port-theme.git themes/

# Deploy locally
$ bash serve.sh

# Build for production
$ hugo --minify --gc
```
