Source code of my blog hosted at [swlacy.com](https://swlacy.com). Built using [Hugo](https://gohugo.io), [TailwindCSS](https://tailwindcss.com/), and my custom theme, [Port](https://github.com/lacysw/hugo-port-theme).

Usage (requires Hugo in path):
```bash
$ git clone https://github.com/lacysw/blog.git && cd blog
$ git clone https://github.com/lacysw/hugo-port-theme.git themes/

# Deploy locally
$ bash serve.sh

# Deploy to Firebase
$ bash deploy.sh

# Update dependencies
$ bash dependencies.sh
```
